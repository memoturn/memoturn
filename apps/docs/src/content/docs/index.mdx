<HeroCard

	code="make dev   # uv run memoturn"
	lang="bash"
>
	Memoturn is an open-source, Docker-based runtime for durable AI agents. Agent actors run
	in-process, each with its own SQLite store; idle actors hibernate to disk and rehydrate
	on demand. Useful at Tier 0 with just a durable filesystem and chat — and it climbs from
	there.
</HeroCard>

Built on FastAPI with a pluggable `Provider`, `Sandbox`, and `Durability` interface — ships with
Anthropic (Claude), Docker, and a SQLite checkpoint engine, and swaps in OpenAI, Ollama, or a
gVisor-isolated sandbox without touching the core. Read the [architecture overview](/architecture/)
for the full request path.
